**Простая и мощная библиотека для отправки email через Gmail и другие SMTP серверы**

## ✨ Особенности

- ✅ **Zero dependencies** - только стандартная библиотека Python
- ✅ **Автоопределение SMTP** по домену email
- ✅ **Поддержка всех популярных почтовых сервисов** (Gmail, Yahoo, Outlook, Mail.ru, Yandex)
- ✅ **HTML письма и вложения**
- ✅ **Массовая рассылка с персонализацией**
- ✅ **Контекстный менеджер** для автоматического управления соединением
- ✅ **Простые и понятные ошибки**
- ✅ **Загрузка конфигурации из JSON или переменных окружения**
- ✅ **Поддержка Python 3.7+**
